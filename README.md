# MarketX - Make InfoBusiness Great Again

Sito ufficiale di MarketX, l'agenzia specializzata in lanci infobusiness milionari.

## Tech Stack

- **Framework**: Astro 4.x
- **UI Components**: React 18
- **Styling**: Tailwind CSS 3.4
- **Fonts**: Google Fonts (Playfair Display + Manrope)
- **Language**: TypeScript + Astro

## Struttura

```
src/
├── components/      # Componenti Astro e React
├── layouts/         # Layout principali
├── pages/           # Pagine (routing automatico)
├── styles/          # CSS globale e Tailwind
```

## Installazione

```bash
npm install
npm run dev      # Dev server su http://localhost:3000
npm run build    # Build per produzione
npm run preview  # Preview della build
```
